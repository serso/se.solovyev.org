About

Java Common Library (JCL) is an open source Java library written due to several reasons:
  1. To provide classes which are missing in JDK (or other well know libraries like Guava)
  2. To fix the mistakes which have been done in JDK (e.g. Cloneable without clone())
  3. To provide different utility classes which can be used quite often

Installation

There are three different ways of installation the library:
  1. Binary jars and source code archives can be downloaded here
  2. Source code can be cloned from github repository (see Source Code section) and built manually
  3. Maven users can just add dependency in pom.xml: 
    <!-- all modules -->
<dependency>
    <groupId>org.solovyev</groupId>
    <artifactId>common-all</artifactId>
    <version>LATEST_VERSION</version>
</dependency>

Component: Core

Description
This component contains several classes and interfaces which can be useful in any Java program.
  1. org.solovyev.common.JObject class which implements Cloneable interface and overrides method clone()
  2. org.solovyev.common.JCloneable interface which extends Cloneable and declares method clone()
  3. org.solovyev.common.JPredicate which is a copy of Guava's com.google.common.base.Predicate
  4. org.solovyev.common.SynchronizedObject which allows to create synchronized objects easily
  5. org.solovyev.common.MutableObject which allows to create mutable object
  6. org.solovyev.common.JBuilder - base interface for all builders
  7. org.solovyev.common.listeners.Listeners - interface for listeners container (object which holds listeners)
  8. org.solovyev.common.interval.Interval which represents typed interval
  9. org.solovyev.common.equals.Equalizer - interface for comparing two objects on equality
Installation
<dependency>
    <groupId>org.solovyev</groupId>
    <artifactId>common-core</artifactId>
    <version>LATEST_VERSION</version>
</dependency>
